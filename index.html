The coding of this website began in early fall 2017 as a project for my Lead 499 course (Nonprofit Organization Internship). For the project, we were to create a website to display various stories of how I embody the traits of the Nonprofit world. When presented with the option to create a wordpress website, I decided it would be much more fun to learn HTML instead. I used brackets for wrtiting and editting my code and github.io for publishing. Much credit goes out to Materialize for providing the JavaScript/CSS that pulled this site together. I will graduate from Kansas State University in Spring of 2018 with degrees in Psychology and Anthropology, and a minor in Leadership Studies, Nonprofit emphasis.
