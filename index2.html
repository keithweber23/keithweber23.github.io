Hi, my name is Keith.

Coding this website began as a project for my Lead 499 course and spanned the majority of Fall, 2017. Utilizing Javascript, CSS, and, of course, HTML I coded it all by hand. The purpose of this website is to provide personal narratives that display some core attributes deemed crucial for working within today's nonprofit industry. Accompanying these naratives are some professional items such as my resume, proposals for chartering a nonprofit and for improving an existing nonprofits volunteer base.
